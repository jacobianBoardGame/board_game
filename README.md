# 籌碼阿瓦隆 | Chip Avalon

基于阿瓦隆的筹码规则扩展与结算系统。

该仓库面向 GitHub Pages 部署，目标是把规则发布、快速介绍、结算工具与后续更新结构化，而不是继续把所有内容塞进单个页面。

## Pages Structure

- `index.html` — 首页 / Home
- `quickstart.html` — 规则介绍 / Quick Start
- `rules.html` — 完整规则 / Full Rules
- `calculator.html` — 简易结算 / Calculator
- `faq.html` — 常见问题 / FAQ
- `changelog.html` — 更新记录 / Changelog
- `assets/css/main.css` — 共用样式
- `assets/js/lang.js` — 双语切换逻辑
- `assets/js/calculator.js` — 结算逻辑

## Project Goals

- 公开规则文本与版本记录
- 提供对新玩家更友好的规则介绍页
- 提供一个可直接演示的轻量结算页面
- 为未来扩展 FAQ、示例局、变体规则和更新日志预留空间

## Suggested Next Steps

- 统一站点 logo 与封面图
- 增加 FAQ 与边界情况说明
- 为结算页加入小盘口与奖池分配细节
- 增加更多变体模式与版本开关
