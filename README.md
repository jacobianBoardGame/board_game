# Avalon Betting Rules Wiki

An unofficial community rule extension inspired by Avalon.

This repository documents a chip-based betting and settlement system designed for Avalon-style social deduction gameplay.  
It is intended as an open, extensible wiki-style project for recording rules, examples, variants, and future revisions.

## Project Scope

This project currently includes:

- core betting and settlement rules
- side-market rules for mission outcomes and identity guesses
- prize pool and entry-fee mechanics
- bilingual presentation support for GitHub Pages
- room for future examples, FAQ, edge cases, and version history
