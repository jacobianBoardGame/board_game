Quick Start

For first-time readers of Chip Avalon: understand the core concepts, match flow, and minimum setup before opening the formal rules page.

1. 核心思路

  • 保留阿瓦隆主局对抗结构
  • 加入筹码结算，让每局结束后有明确收付结果
  • 允许发车盘、身份盘等小盘口作为附加互动层

1. Core idea

  • Keeps the main team-vs-team structure of Avalon
  • Adds chip settlement to produce clear pay/receive results after each game
  • Allows side markets such as mission bets and identity bets for extra interaction

2. 三个基础参数

i = 基础单位，j = 主局系数，z = 入场费系数

默认示例：i = 1，j = 5，z = 2。

2. Three base parameters

i = base unit, j = main-game multiplier, z = entry-fee multiplier

Default example: i = 1, j = 5, z = 2.

3. 一局怎么结算

  • 先算主局胜负
  • 再算特殊事件加减分
  • 小盘口单独平衡
  • 最后合并成每人总分与收付结果

3. How one game settles

  • Calculate the main-game win/loss first
  • Then add special-event bonuses and penalties
  • Settle side markets separately
  • Combine everything into each player's final total and payment result

4. 最小使用方式

  1. 确定 i, j, z
  2. 每位玩家支付入场费进入公共池
  3. 完成一局阿瓦隆
  4. 根据胜负和特殊事件结算主局
  5. 如有小盘口，再分别结算
  6. 输出每位玩家最终应收 / 应付

4. Minimum operating flow

  1. Set i, j, and z
  2. Each player pays the entry fee into the shared pool
  3. Play one game of Avalon
  4. Settle the base game using win/loss and special events
  5. Settle side markets if any were opened
  6. Output each player's final receive/pay result
