Chip Avalon

A chip-based rule extension and settlement system for Avalon. Built for public publishing, revision tracking, social media distribution, and future expansion.

GitHub Pages Ready Bilingual Extensible
Quick Start Full Rules Calculator

这是什么

籌碼阿瓦隆把主局胜负、特殊结算、小盘口与公共奖池整理为一套可复用的规则框架。

What it is

Chip Avalon organizes win/loss settlement, special events, side markets, and prize-pool logic into a reusable framework.

站点结构

首页负责品牌展示；规则介绍页面向新玩家；完整规则页保存正式文本；结算页提供快速计算。

Site structure

The home page handles branding, Quick Start targets new players, Rules stores the formal text, and Calculator handles settlement.

未来扩展

已预留 FAQ、更新日志、边界案例、变体规则、示例局与更多自动化结算功能。

Future expansion

FAQ, changelog, edge cases, rule variants, sample games, and richer automated settlement features can be added later.

项目定位

  • 公开规则文本与版本记录
  • 提供低门槛的在线说明与结算功能
  • 为未来扩展预留结构，不把所有内容塞进单个页面

Project goals

  • Publish the rule text and version history
  • Provide low-friction online explanation and settlement tools
  • Leave room for future growth instead of forcing everything into one page
